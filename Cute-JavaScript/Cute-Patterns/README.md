设计模式是可重用的用于解决软件设计中一般问题的方案，学习好一些设计模式，不仅让我们站在巨人的肩膀上，获得前人所有的经验，保证我们以优雅的方式组织我们的代码，满足我们解决问题所需要的条件。

JavaScript是一种弱类型的、动态的、基于原型的语言，这使得它能很容易的实现一些设计模式。   

本文我们将介绍九种常见的设计模式，看看在JavaScript中如何实现这些设计模式。
